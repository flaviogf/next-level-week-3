## :thinking: Como utilizar

### Backend

```sh
cd backend

yarn

yarn typeorm migration:run

yarn start
```

### Frontend

```sh
cd frontend

yarn

yarn start
```

### Mobile

```sh
cd mobile

yarn

yarn start
```
